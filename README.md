<h1>Frontend Mentor - Intro Component with Sign Up Form</h1>

<h2>Overview</h2>

<p>This project is a solution to the <a href="https://www.frontendmentor.io/challenges/intro-component-with-signup-form-5cf91bd49edda32581d28fd1">Intro component with sign-up form challenge on Frontend Mentor</a>. It features a responsive sign-up form with client-side validation built using HTML, CSS, and JavaScript. The goal was to practice and learn new concepts in front-end development, including form handling, validation, and responsive design.</p>

<h2>Features</h2>

<ul>

<li>Responsive design that adapts to desktop and mobile devices.</li>

<li>Client-side form validation for first name, last name, email, and password fields.</li>

<li>Real-time validation feedback with error messages, success indicators, and icons.</li>

<li>Password requirements: At least 8 characters, including uppercase, lowercase, and a number.</li>

<li>Custom styling with CSS variables for colors and fonts (using Poppins font from Google Fonts).</li>

<li>Background image and layout optimized for a clean, modern look.</li>

</ul>
